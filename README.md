# ⏱️ Pomodoro Timer App

A simple and clean Pomodoro timer built using Python and Tkinter.  
Perfect for staying focused and boosting productivity.

## Features
- ✅ 25-minute work sessions
- ✅ Short & long breaks
- ✅ Visual countdown timer
- ✅ Session tracker (✔️)
